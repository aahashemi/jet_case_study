# Project Setup

## Installation
Note: Docker Engine must be active and running before executing the build script#

1. Clone the repository:
    ```bash
    git clone https://github.com/aahashemi/jet_case_study.git
    cd jet_case_study
    ```

2. Run the build script:
    ```bash
    bash build.sh
    ```

3. Access the application:
    Open your browser and navigate to `http://localhost:8080`
